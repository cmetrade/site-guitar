> Um site interativo sobre guitarra e teoria musical, construído com HTML, CSS e JavaScript puro.

## ✨ Funcionalidades

- **Hero animado** — Silhueta SVG de guitarra com cordas que vibram e reagem ao clique
- **Catálogo de guitarras** — Cards ilustrados de Violão Clássico, Guitarra Elétrica e Contrabaixo
- **Quiz de acordes** — Identifique acordes no braço da guitarra com feedback em tempo real
- **Teoria musical** — Seção educativa sobre notas, semitons, tríades e progressões
- **Afinador visual** — Simulador de afinação corda por corda com agulha animada

---

## 🛠️ Tecnologias

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura semântica |
| CSS3 | Animações, Grid, variáveis CSS |
| JavaScript (Vanilla) | Quiz interativo, afinador, cursor customizado |
| Google Fonts | Playfair Display · Cormorant Garamond · DM Mono |
| SVG | Ilustrações de guitarras e braço do instrumento |

## 🚀 Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/sixstrings.git
   ```

2. Abra o arquivo no navegador:
   ```bash
   cd sixstrings
   open guitarra.html
   ```

Não há dependências externas nem processo de build — é só abrir e tocar. 🎵

## 🎨 Design

Paleta inspirada em madeira, âmbar e jazz — tipografia editorial com **Playfair Display** para títulos e **DM Mono** para elementos de interface. Cursor personalizado, micro-animações em hover e transições suaves em todas as interações.
